# 🏐 Haikyuu - Karasuno Quiz

## 📖 Sobre o Projeto

Este projeto foi desenvolvido como Projeto Individual da SPTech.

O objetivo é apresentar informações sobre o anime **Haikyuu!!** e o time **Karasuno**, permitindo que os usuários conheçam melhor os personagens e participem de um quiz interativo para descobrir o quanto combinam com os famosos "Corvos" de Karasuno.

---

## ✨ Funcionalidades

* Página inicial com apresentação do projeto
* Informações sobre o time Karasuno
* Exibição dos principais personagens
* Quiz interativo com perguntas sobre o universo de Haikyuu
* Sistema de cadastro e login
* Dashboard com resultados
* Integração com banco de dados

## 🚀 Como Executar

###Clonar o repositório

```bash
git clone https://github.com/LesleyOliveira/Projeto_Individual.git
```

### Entrar na pasta do projeto

```bash
cd Projeto_Individual
```

### Instalar dependências

```bash
npm i
```

### Iniciar aplicação

```bash
npm start
```
